# OreTrack-Frontend
Frontend source code for OreTrack. Oretrack is an application for rescue. Developed by Oreoreo team during JunctionxHanoi Hackathon 2023


## Domain

| Domain                | Name       |
|-----------------------|------------|
| http://localhost:5173 | Local      |
| https://oretrack.life | Production |

## Project Setup

```sh
cd oretrack-fe
```

```sh
npm install
```

```sh
npm run format
```

```sh
npm run dev
```
